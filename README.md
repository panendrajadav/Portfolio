# Panendra Jadav - Portfolio

A modern, responsive portfolio website showcasing my work in AI/ML, blockchain, and full-stack development.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Framer Motion

## Getting Started

To run this project locally:

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd <YOUR_PROJECT_NAME>

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Features

- Responsive design
- Dark/Light theme toggle
- Animated components
- Dynamic GitHub activity
- Project showcase
- Contact information
